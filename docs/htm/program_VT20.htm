Program för våren 2020

OBS! SENIORSCHACK STOCKHOLM AVSLUTAR VÅRENS VERKSAMHET OMGÅENDE.

Efter att Sveriges Schackförbund rekommenderat att alla evenemang för personer i riskgruppen ställs in och Stockholm Schackförbund beslutat att hålla Salongerna stängda åtminstone till 30 april har Seniorschacks styrelse beslutat att avsluta vårens verksamheter. Vårt förslag är att Seniorserien startar om på nytt i höst med samma gruppindelning (plus eventuellt nyinkomna) samt att KM inställs och att vi istället satsar på att genomföra ett Stockholmsmästerskap till hösten. Inte heller kommer tisdagarnas teoriverksamhet, våravslutning, Schack till sjöss, lagmatch mot Göteborg och/eller Uppsala att genomföras i vår regi.
