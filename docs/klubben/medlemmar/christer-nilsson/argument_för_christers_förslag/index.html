Vad rekommenderas för att kunna uppdatera?

  • Visual Studio Code från Microsoft
    • Markdown Preview
    • Integration med Github
    • Live Server - man ser ändringarnas resultat direkt i bläddraren.
    • Run On Save - när man sparar en markdown-fil skapas automatiskt html-filen.
  • Python - Världens populäraste programmeringsspråk.
  • makeAll.exe - ett tvåsidors pythonscript som skapar nödvändiga html-filer.
  • Chrome - browsern från Google
